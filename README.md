# Admin_Crud_Creator
Create admin crud module from just providing namespace and module name.

# Features
 -- New module have Admin menu 
 
 -- New grid with mass action delete 
 
 -- Add data button 
 
 -- Edit and delete action in line data


# Settings
You need to install this module.
